# Hyprbuntu

A modern, minimalist Linux distribution configuration combining Hyprland window manager with Ubuntu's stability and familiarity.

## 🎯 Overview

Hyprbuntu brings together the best of both worlds:
- **Ubuntu** - A reliable, well-supported Linux foundation
- **Hyprland** - A cutting-edge, dynamic tiling window manager

Perfect for developers, power users, and enthusiasts who want a lightweight, keyboard-driven desktop experience.

## ✨ Features

- **Dynamic Tiling Window Manager** - Hyprland's powerful tiling capabilities
- **GPU Acceleration** - Modern graphics rendering for smooth animations
- **Keyboard-Centric Workflow** - Efficient window management without the mouse
- **Minimal Resource Usage** - Lean and fast performance
- **Ubuntu Foundation** - Familiar package management and ecosystem
- **Highly Customizable** - Extensive configuration options

## ⚙️ Configuration

Configuration files are located in `~/.config/hypr/`. Key files include:

- `hyprland.conf` - Main Hyprland configuration
- `keybindings.conf` - Custom keyboard shortcuts
- `colors.conf` - Color scheme and theme settings

## 🎨 Customization

Edit your configuration files to customize:
- Window layouts and gaps
- Keyboard shortcuts
- Color schemes
- Monitor settings
- Application rules
